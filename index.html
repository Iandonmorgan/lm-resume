Celebrity Tribute: Keanu Reeves (Group Project)

As part of a class assignment, I built a tribute site to Keanu Reeves with 3 teammates. This was our first group project at Nashville Software School, and we used HTML, CSS, Flexbox, Visual Studio Code, Git, Github, and http-server to build the website.

Lessons I learned during this project include how important it is to plan and communicate with teammates. This was my first exposure to git and Github, but my comfort level has grown exponentially with this project. I feel it is extremely important with group projects to outline plans and paths to allow for asynchronous and synchronous conversations throughout your project, and to define what conversations should happen on each path.

Link to project: github.com/iandonmorgan/celebrity-tribute-keanu-reeves
